 ### How to run
   + copy app
  ```
  $ git clone https://github.com/veronga/jop
  ```

   + open downloaded folder
 
  ```
  $  cd jop
  ```

   + install necessary packages
  ```
  $  npm install
  ```
  
   + install PODs
  ```
  $  cd ios
  ```
  ```
  $  pod install
  ```
  
   + Now let's launch the app, but don't forget to log out of the ios folder
   
  ```
  $  cd ..
  ```
  ```
  $  react-native run-[ios|android]
  ```
